ИНСТРУКЦИИ ЗА СЪЗДАВАНЕ НА PIVOT ТАБЛИЦА В EXCEL
==================================================

Файлът данни-обобщени-таблици-11а.xlsx съдържа:
- Колони A-G: Оригинални данни (140 записа)
- Колони K-Q: Примерен резултат (статична таблица)

НАСТРОЙКИ НА ПРИМЕРНАТА PIVOT ТАБЛИЦА:
---------------------------------------

1. РЕДОВЕ (Rows): Категория
2. КОЛОНИ (Columns): Регион
3. СТОЙНОСТИ (Values): Сума от "Продажби"
4. ФИЛТРИ (Filters): Няма

═══════════════════════════════════════════════════════════════════

СТЪПКА 1: ПОДГОТОВКА НА ДАННИТЕ
---------------------------------

1. Отвори файла данни-обобщени-таблици-11а.xlsx
2. Кликни на клетка A1 (заглавието)
3. Използвай Ctrl+Shift+End за да маркираш всички данни
   (Или маркирай ръчно от A1 до G141)

ВАЖНО: Данните трябва да имат:
- Заглавия в първия ред
- Без празни редове или колони
- Всички колони с еднакъв формат

═══════════════════════════════════════════════════════════════════

СТЪПКА 2: СЪЗДАВАНЕ НА PIVOT ТАБЛИЦА
--------------------------------------

1. С маркирани данни, отиди на лентата:
   Insert → PivotTable

2. Появява се диалогов прозорец "Create PivotTable":
   - Table/Range: $A$1:$G$141 (автоматично)
   - Choose where: New Worksheet (препоръчително)
   
3. Натисни OK

4. Отваря се нов лист с празна Pivot таблица
   и панел "PivotTable Fields" отдясно

═══════════════════════════════════════════════════════════════════

СТЪПКА 3: КОНФИГУРИРАНЕ НА ПОЛЕТАТА
-------------------------------------

Вдясно виждаш панел с две части:
- Горе: списък на всички налични полета
- Долу: 4 области (Filters, Columns, Rows, Values)

ДЕЙСТВИЯ:

1. От горния списък ВЛАЧИ "Категория" в областта "Rows"
   (Или маркирай "Категория" и избери "Add to Rows")

2. ВЛАЧИ "Регион" в областта "Columns"

3. ВЛАЧИ "Продажби" в областта "Values"

РЕЗУЛТАТ:
Автоматично се появява таблица с категории по редове
и региони по колони, със суми на продажбите.

═══════════════════════════════════════════════════════════════════

СТЪПКА 4: НАСТРОЙКА НА ИЗЧИСЛЕНИЯТА
-------------------------------------

По подразбиране Excel използва SUM за числа.
Ако искаш да провериш или промениш:

1. В областта "Values" виждаш "Sum of Продажби"
2. Кликни на стрелката до него
3. Избери "Value Field Settings..."

4. В диалоговия прозорец:
   - Summarize value field by: Sum (Сума)
   - Custom Name: можеш да смениш на "Продажби"
   
5. Кликни на "Number Format..." за форматиране:
   - Избери "Number"
   - Decimal places: 0
   - Use 1000 separator: ✓
   
6. Натисни OK два пъти

═══════════════════════════════════════════════════════════════════

СТЪПКА 5: ФОРМАТИРАНЕ НА ТАБЛИЦАТА
------------------------------------

1. Стил на таблицата:
   - Кликни върху Pivot таблицата
   - Design → PivotTable Styles
   - Избери стил (напр. "Medium 9")

2. Опции за показване:
   - Design → Report Layout → Show in Tabular Form
   - Design → Subtotals → Do Not Show Subtotals

3. Показване на общи суми:
   - Design → Grand Totals → On for Rows and Columns

═══════════════════════════════════════════════════════════════════

ДОПЪЛНИТЕЛНИ ФУНКЦИИ
---------------------

А) ФИЛТРИРАНЕ

1. Добавяне на филтър:
   - Влачи "Продукт" в областта "Filters"
   - Над таблицата се появява падащо меню
   - Избери конкретни продукти за показване

2. Филтриране в редове/колони:
   - Кликни на стрелката до "Категория" или "Регион"
   - Премахни отметки от нежеланите стойности

Б) SLICER (ВИЗУАЛЕН ФИЛТЪР)

1. Кликни върху Pivot таблицата
2. PivotTable Analyze → Insert Slicer
3. Избери "Категория" и "Регион"
4. Натисни OK
5. Появяват се панели с бутони за филтриране

В) ГРУПИРАНЕ ПО ДАТИ

1. Добави "Дата" в Rows (вместо или заедно с "Категория")
2. Десен клик на дата в таблицата → Group
3. Избери: Months, Quarters или Years
4. Натисни OK

Г) ПРОЦЕНТНО ПОКАЗВАНЕ

1. Кликни на "Sum of Продажби" в Values
2. Value Field Settings
3. Show Values As → % of Grand Total
4. OK

Д) ДОПЪЛНИТЕЛНО ПОЛЕ (БРОЙ)

1. Влачи "Продажби" още веднъж в Values
2. Ще се появи "Sum of Продажби2"
3. Кликни на него → Value Field Settings
4. Summarize by: Count
5. Custom Name: "Брой продажби"

═══════════════════════════════════════════════════════════════════

АКТУАЛИЗИРАНЕ НА ДАННИТЕ
--------------------------

Ако промениш оригиналните данни:

1. Кликни върху Pivot таблицата
2. Десен клик → Refresh
   (Или натисни Alt+F5)

Ако добавиш нови редове:

1. Десен клик на таблицата → PivotTable Options
2. Data → Change Data Source
3. Избери новия диапазон (напр. A1:G200)
4. OK

═══════════════════════════════════════════════════════════════════

КЛАВИШНИ КОМБИНАЦИИ
---------------------

Alt+F5         - Refresh на Pivot таблицата
Ctrl+A         - Избор на цялата таблица
Alt+D+P        - Създаване на нова Pivot таблица (стар вариант)

═══════════════════════════════════════════════════════════════════

ЧЕСТИ ГРЕШКИ И РЕШЕНИЯ
------------------------

Проблем: "The PivotTable field name is not valid"
Решение: Провери дали всички колони имат заглавия

Проблем: Таблицата не се актуализира
Решение: Натисни Refresh (Alt+F5)

Проблем: Виждам "Count" вместо "Sum"
Решение: Колоната вероятно има текст - провери форматите

Проблем: Липсват данни в таблицата
Решение: Провери филтрите и Slicers

═══════════════════════════════════════════════════════════════════

ЕКСПОРТИРАНЕ НА РЕЗУЛТАТА
---------------------------

За статична копие на таблицата:

1. Маркирай цялата Pivot таблица
2. Ctrl+C (Copy)
3. Отиди на нов лист
4. Десен клик → Paste Special → Values
5. Получаваш статична таблица без връзка към данните
